# Communication Channel Monitoring
There are two logs generated by the Communication Channel logging, detailed and summary.   
## Detailed Log Contents   
The following data is reported if a non-null and not an empty string.   
Name   
Channel Id    
Channel Status   
Short Log    
ControlData    
Processing Errors   
Party   
Component    
Adapter Type   
Direction   
Number of Nodes    
Cluster Data:    
&nbsp;&nbsp;&nbsp;Name    
&nbsp;&nbsp;&nbsp;NodeId       
&nbsp;&nbsp;&nbsp;Cluster Status   
&nbsp;&nbsp;&nbsp;Cluster Short Log    
Processing Detail    
&nbsp;&nbsp;&nbsp;Name   
&nbsp;&nbsp;&nbsp;Type   
&nbsp;&nbsp;&nbsp;Explanation   
&nbsp;&nbsp;&nbsp;Timestamp   
&nbsp;&nbsp;&nbsp;Message ID
